# extragalactic-proton-propagation

This repository contains the code developed during my undergraduate research with Astroparticle Physics Group (APOEMA) at University of São Paulo.

The central point is solve the problem of proton propagation through an expanding Universe, filled by cosmic microwave background isotropic photon field. The interactions taken into account were: photo-pion and photo-pair production. As well as adiabatic energy loss by Universe expansion. 

I propagated protons emitted by an one source simple model and compute the Earth measured flux. So I was able to reproduce propagation effects on end of ultra-high energy cosmic rays energy spectrum.
